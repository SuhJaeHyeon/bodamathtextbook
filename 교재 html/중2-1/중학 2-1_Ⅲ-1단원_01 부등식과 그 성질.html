개념 01
부등식이란 무엇인가? — (1) 부등식
부등식 : 부등호 $>$, $<$, $\geq$, $\leq$를 사용하여 두 수 또는 두 식 사이의 대소 관계를 나타낸 식
  예) $5>2$,  $x<5$,  $x+1\geq2$,  $2-3x\leq5-7x$
좌변 : 부등호 왼쪽의 식
우변 : 부등호 오른쪽의 식
양변 : 좌변과 우변을 함께 이르는 말
예제 01
다음 중 부등식인 것을 모두 고르시오. (정답 2개)
①  $7-3=4$ ②  $2a>-1$ ③  $3x+5$ ④  $x-4\leq9$ ⑤  $5y-(2y+1)$
참고
등식과의 차이
  • 등식 : 등호(=)로 두 식이 같음 → $x+1=2$
  • 부등식 : 부등호로 크기 차이 → $x+1>2$
풀이 TIP
  • 부등호($>,<,\geq,\leq$) 있으면 → 부등식
  • 등호(=)만 있으면 → 등식
  • 둘 다 없으면 → 단순 식

부등식이란 무엇인가? — (2) 부등식의 표현
$a>b$$a<b$$a\geq b$$a\leq b$
a는 b보다 크다a는 b보다 작다 크거나 같다작거나 같다
b 초과b 미만b 이상b 이하
—— b보다 작지 않다b보다 크지 않다
예) x는 4보다 크다 → $x>4$  /  x는 −2 미만 → $x<-2$
x는 $\frac{1}{3}$ 이상 → $x\geq\frac{1}{3}$  /  x는 0보다 크지 않다 → $x\leq0$
예제 02
다음 문장을 부등식으로 나타낸 것으로 옳지 않은 것은?
①  x의 3배에 5를 더한 수는 7보다 크다. → $3x+5>7$ ②  x에서 4를 뺀 수의 2배는 17보다 크지 않다. → $2(x-4)\leq17$ ③  한 자루에 600원인 연필 x자루의 가격은 5000원 미만이다. → $600x<5000$ ④  닭 x마리와 개 y마리의 전체 다리 수는 55보다 많다. → $4x+2y>55$ ⑤  시속 5 km로 x시간 걸은 거리는 20 km 이상이다. → $5x\geq20$
참고
이상·이하 → $\geq$, $\leq$
초과·미만 → $>$, $<$
"크지 않다" → $\leq$
"작지 않다" → $\geq$
풀이 TIP
  • 한글 표현을 부등호로 변환
  • 이상·이하 → $\geq$, $\leq$
  • 초과·미만 → $>$, $<$
  • 실생활 문제 : 단위 확인!

부등식과 그 성질 003
개념 03
부등식은 어떤 성질을 가질까?
(1) 같은 수를 더하거나 빼는 경우
$a<b$이면 $a+c<b+c$,  $a-c<b-c$  →  방향 그대로
(2) 같은 양수를 곱하거나 나누는 경우
$a<b$, $c>0$이면 $ac<bc$,  $\dfrac{a}{c}<\dfrac{b}{c}$  →  방향 그대로
(3) 같은 음수를 곱하거나 나누는 경우
$a<b$, $c<0$이면 $ac>bc$,  $\dfrac{a}{c}>\dfrac{b}{c}$  →  방향 반드시 반대로 ⚠️
예제 04
$a<b$일 때, 다음 중 옳지 않은 것은?
①  $a+2<b+2$ ②  $-5a>-5b$ ③  $6-a<6-b$ ④  $2a-3<2b-3$ ⑤  $-\dfrac{a}{3}+1>-\dfrac{b}{3}+1$
참고
$2<4$에서 :
$+2$→$4<6$ 그대로
$-2$→$0<2$ 그대로
$\times2$→$4<8$ 그대로
$\times(-2)$→$-4>-8$ 반대
$\div(-2)$→$-1>-2$ 반대
풀이 TIP
  • 양수 곱/나누기 → 그대로
  • 음수 곱/나누기 → 반대!
  • 뺄셈 = 음수 덧셈 변환
  • $6-a=(-1)\times a+6$

개념 03
부등식의 성질 — 조건에서 관계 유도하기
주어진 조건 식을 변형하여 $a$와 $b$의 대소 관계를 먼저 파악한 뒤, 성질을 적용한다.
예) $-3a+2>-3b+2$ 일 때
양변에서 2를 뺌 : $-3a>-3b$
→ 양변을 $-3$으로 나눔 (음수, 방향 반대) : $a<b$
→ 이후 $a<b$를 이용하여 각 식에 성질 적용
예제 05
$-3a+2>-3b+2$일 때, 다음 □ 안에 알맞은 부등호를 써넣으시오.
(1) $a$ □ $b$ (2) $4a-3$ □ $4b-3$ (3) $5-7a$ □ $5-7b$ (4) $-\dfrac{a}{5}+\dfrac{1}{2}$ □ $-\dfrac{b}{5}+\dfrac{1}{2}$
참고
조건 식 → 기본 대소관계($a$ □ $b$) 파악 → 성질 적용 순서로!
풀이 TIP
  • 조건에서 $a$, $b$ 관계 먼저 파악
  • $a$, $b$에 곱해지는 숫자가 양수인지 음수인지가 부등호 방향을 결정한다!
정답 (1) $<$  (2) $<$  (3) $>$  (4) $>$

보충 개념
부등식의 성질을 이용하여 식의 값의 범위 구하기
$a<x<b$일 때 $cx+d$의 범위 구하기 :
 ❶ 각 변에 $c$를 곱함 — $c>0$이면 방향 그대로, $c<0$이면 방향 반대
 ❷ ❶의 결과 각 변에 $d$를 더함
예) $-2<x<1$일 때 :
 • $3x+5$ :
  $x$에 $\times3$ → $-6<3x<3$
  $3x$에 $+5$ → $-1<3x+5<8$
 • $-3x+5$ :
  $x$에 $\times(-3)$ → $-3<-3x<6$
  $-3x$에 $+5$ → $2<-3x+5<11$
예제 06
$-1<a<3$일 때, 다음 값의 범위를 구하시오.
(1) $3a-7$ (2) $-2a+1$

양수 계수 : 방향 그대로
음수 계수 : 방향 반대
음수 곱 후에는 "작은 수 < 식 < 큰 수"로 정리
풀이 TIP
  • 계수 부호 먼저 확인!
  • 양수 → 방향 불변
  • 음수 → 방향 반대!
  • 곱셈 후 덧셈 순서 지키기
  • 등호 포함 여부 주의

부등식과 그 성질 006
한눈에 보는 개념 정리
01 부등식이란?
부등식 : 부등호($>$, $<$, $\geq$, $\leq$)를 사용하여 두 수 또는 두 식 사이의 대소 관계를 나타낸 식
  • 좌변 : 부등호 왼쪽의 식   우변 : 부등호 오른쪽의 식   양변 : 좌변과 우변을 함께 이르는 말
  • 해 : 부등식을 참이 되게 하는 미지수의 값  /  부등식을 푼다 : 해를 구하는 것
02 부등식의 표현
$a>b$$a<b$$a\geq b$$a\leq b$
$a$는 $b$보다 크다$a$는 $b$보다 작다 크거나 같다작거나 같다
$b$ 초과$b$ 미만$b$ 이상$b$ 이하
—— $b$보다 작지 않다$b$보다 크지 않다
03 부등식의 성질
  • $a<b$이면 $a+c<b+c$, $a-c<b-c$  (어떤 수 $c$에 대해 성립)
  • $a<b$이고 $c>0$이면  $ac<bc$,  $\dfrac{a}{c}<\dfrac{b}{c}$
  • $a<b$이고 $c<0$이면  $ac>bc$,  $\dfrac{a}{c}>\dfrac{b}{c}$  (부등호 방향이 바뀜!)
