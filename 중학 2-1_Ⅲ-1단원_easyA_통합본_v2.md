## 【단원 01】 부등식과 그 성질

---

### 개념 1 | 부등식이란 무엇인가?

#### (1) 부등식

- **부등식**: 부등호 `>`, `<`, `≥`, `≤`를 사용하여 두 수 또는 두 식 사이의 대소 관계를 나타낸 식
- 예: `5 > 2`,  `x < 5`,  `x + 1 ≥ 2`,  `2 − 3x ≤ 5 − 7x`

**각 부분의 명칭**
- **좌변**: 부등호의 왼쪽에 있는 식
- **우변**: 부등호의 오른쪽에 있는 식
- **양변**: 좌변과 우변을 함께 이르는 말

**[참고] 등식과 부등식의 차이**
- 등식: 등호(=)로 두 식이 같음을 나타냄 → `x + 1 = 2`
- 부등식: 부등호로 두 식의 크기 관계를 나타냄 → `x + 1 > 2`

---

> **예제 01 | 부등식의 뜻**
>
> 다음 중 부등식인 것을 모두 고르시오. **(정답 2개)**
>
> ① `7 − 3 = 4`　　② `2a > −1`　　③ `3x + 5`　　④ `x − 4 ≤ 9`　　⑤ `5y − (2y + 1)`
>
> **[정답]** ②, ④
>
> **[풀이]**
> - 부등식이란 부등호(`>`, `<`, `≥`, `≤`)가 포함된 식이다.
> - ① `7−3=4`: 등호(=)만 있으므로 **등식** (부등식 ✕)
> - ② `2a>−1`: 부등호 `>` 포함 → **부등식 ✓**
> - ③ `3x+5`: 등호도 부등호도 없음 → **단순 식** (부등식 ✕)
> - ④ `x−4≤9`: 부등호 `≤` 포함 → **부등식 ✓**
> - ⑤ `5y−(2y+1)`: 등호도 부등호도 없음 → **단순 식** (부등식 ✕)

#### (2) 부등식의 표현

| 기호 | `a > b` | `a < b` | `a ≥ b` | `a ≤ b` |
|------|---------|---------|---------|---------|
| 표현 ① | a는 b보다 크다 | a는 b보다 작다 | a는 b보다 크거나 같다 | a는 b보다 작거나 같다 |
| 표현 ② | a는 b 초과이다 | a는 b 미만이다 | a는 b 이상이다 | a는 b 이하이다 |
| 표현 ③ | — | — | a는 b보다 작지 않다 | a는 b보다 크지 않다 |

**[참고]**
- `a ≥ b` : `a > b` 또는 `a = b`를 모두 포함
- `a ≤ b` : `a < b` 또는 `a = b`를 모두 포함

**예**
- x는 4보다 크다 → `x > 4`
- x는 −2 미만이다 → `x < −2`
- x는 1/3보다 크거나 같다 → `x ≥ 1/3`
- x는 0보다 크지 않다 → `x ≤ 0`

### 개념 2 | 부등식의 해란 무엇인가?

#### (1) 부등식의 해와 풀기

- **부등식의 해**: 부등식을 참이 되게 하는 미지수의 값
- **부등식을 푼다**: 부등식의 해를 모두 구하는 것

**참·거짓 판단 기준**
- 미지수에 수를 대입했을 때, 좌변과 우변의 대소 관계가 주어진 부등호 방향과 **같으면 → 참** (해이다)
- 주어진 부등호 방향과 **다르면 → 거짓** (해가 아니다)

**예** x의 값이 1, 2, 3일 때, `2x + 1 > 4`를 풀어 보자.

| x | 좌변의 값 | 대소 비교 | 우변의 값 | 참·거짓 |
|---|----------|----------|----------|---------|
| 1 | `2×1+1=3` | `<` | 4 | 거짓 |
| 2 | `2×2+1=5` | `>` | 4 | 참 |
| 3 | `2×3+1=7` | `>` | 4 | 참 |

→ 부등식 `2x + 1 > 4`의 해: **2, 3**

### 개념 3 | 부등식은 어떤 성질을 가질까?

#### (1) 같은 수를 더하거나 빼는 경우

- `a < b`이면 → `a + c < b + c`,  `a − c < b − c`
- 부등호의 방향: **그대로 유지**

#### (2) 같은 양수를 곱하거나 나누는 경우

- `a < b`, `c > 0`이면 → `ac < bc`, `a/c < b/c`
- 부등호의 방향: **그대로 유지**

#### (3) 같은 음수를 곱하거나 나누는 경우

- `a < b`, `c < 0`이면 → `ac > bc`, `a/c > b/c`
- 부등호의 방향: **반드시 반대로 바뀜** ⚠️

**[참고]** 위 세 가지 성질은 `<` 대신 `≤`, `>` 대신 `≥`일 때도 동일하게 성립한다.

---

**[설명]** `2 < 4`를 예로 확인해 보자.

| 연산 | 결과 | 부등호 방향 |
|------|------|------------|
| 양변에 2를 더함 | `4 < 6` | 그대로 |
| 양변에서 2를 뺌 | `0 < 2` | 그대로 |
| 양변에 2를 곱함 | `4 < 8` | 그대로 |
| 양변을 2로 나눔 | `1 < 2` | 그대로 |
| 양변에 **−2를 곱함** | `−4 > −8` | **반대로** |
| 양변을 **−2로 나눔** | `−1 > −2` | **반대로** |

**[보충] 부등식의 성질을 이용하여 식의 값의 범위 구하기**

`a < x < b`일 때 `cx + d`의 범위 구하는 방법:

- ❶ 각 변에 `c`를 곱한다.
  - `c > 0`이면 → `ac < cx < bc` (방향 그대로)
  - `c < 0`이면 → `ac > cx > bc` (방향 바뀜)
- ❷ ❶의 결과 각 변에 `d`를 더한다.

**예** `−2 < x < 1`일 때, `3x + 5`의 값의 범위를 구해 보자.
- 각 변에 3을 곱함 (양수 → 방향 그대로): `−6 < 3x < 3`
- 각 변에 5를 더함: `−1 < 3x + 5 < 8`
- ∴ **`−1 < 3x + 5 < 8`**

> **[풀이]**
> - 각 변에 −2를 곱함 (음수 → 부등호 방향 반대): `−6 < −2a < 2`
> - 각 변에 1을 더함: `−5 < −2a + 1 < 3`
> - ∴ `−5 < −2a + 1 < 3`
