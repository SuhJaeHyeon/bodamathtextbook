# 보다수학 교재 교육과정 분석

> **출처**: `resource/22개정 개념원리 중학 수학 1-1 ~ 3-2 수업용 PPT` 전체 51개 PDF 분석
> **추출 방법**: 각 PDF의 소단원 표지(녹색 배너 + 번호 뱃지)를 시각 분석으로 식별
> **마지막 업데이트**: 2026-04-28

본 문서는 보다수학 학원 교재 제작의 기준이 되는 **22개정 교육과정 - 개념원리 중학 수학** 의 학년별 단원 구조를 정리한 것입니다. 학년 → 대단원 → 중단원 → 소단원의 4단계로 구성되어 있으며, 소단원 단위가 교재 한 회차(HTML 파일 1개)의 기본 단위가 됩니다.

---

## 명명 규칙

git에 커밋되는 교재 HTML 파일은 다음 규칙을 따릅니다:

```
중학 [학년-학기]_[대단원]-[중단원]단원_[소단원번호] [소단원명].html
```

예시:
- `중학 2-1_Ⅲ-1단원_01 부등식과 그 성질.html`
- `중학 2-1_Ⅲ-1단원_02 일차부등식의 풀이.html`

대단원은 **로마숫자**(Ⅰ, Ⅱ, Ⅲ ...) 또는 **아라비아숫자**(1, 2, 3 ...)로 표기되며, 학년별 PDF 파일 명명 관행을 따릅니다.

## 작업 분배 시 참고사항

1. **PDF 파일과 중단원의 매핑**: 대부분의 PDF는 중단원 1개에 대응하지만, 일부 예외가 있습니다.
   - `중 1-2 PPT 1-2단원.pdf`: "위치관계"와 "평행선" 두 중단원 통합 수록
   - `중 1-2 PPT 1-3단원.pdf`: "작도와 합동" — 작도 중단원과 합동 중단원 통합 가능

2. **일부 PPT의 표제 오기**: 자료 PDF 일부 페이지에서 슬라이드 상단 표제가 본문 내용과 일치하지 않는 경우가 발견되었습니다(예: 중1-1 4-1단원 일부 페이지). 교재 제작 시 표제는 본 문서의 분석 결과를 기준으로 하시면 됩니다.

3. **표준 명명 예시**:
   - `중학 1-1_1-1단원_01 소인수분해.html`
   - `중학 2-1_Ⅴ-1단원_03 일차함수의 그래프의 성질.html`
   - `중학 3-2_Ⅱ-2단원_02 원과 사각형.html`
